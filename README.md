⌚ Luxury Watch Analysis – Sales, Customer & Market Insights Dashboard

This project presents an interactive Power BI dashboard designed to analyze luxury watch market performance, customer behavior, and brand sales trends. The dashboard transforms raw sales data into meaningful business insights to help stakeholders make data-driven decisions.

It provides a clear overview of revenue performance, customer segmentation, brand comparison, and city-wise sales trends in the luxury watch industry.

📊 Key Features

Brand Performance Analysis – Compare revenue generated by different luxury watch brands.

City-wise Sales Insights – Identify top-performing markets and locations.

Customer Segmentation – Understand purchasing behavior and customer distribution.

Interactive Navigation – Dashboard sections such as Home, Segment, Customer, and Insights for better exploration.

Business Intelligence Visualization – Clean and professional visuals for strategic decision-making.

🛠 Tools & Technologies

Power BI – Data visualization & dashboard creation

DAX – Measures and calculated insights

Data Modeling – Structured relationships between datasets

Data Analysis – Extracting meaningful insights from sales data

🎯 Project Objective

The goal of this project is to demonstrate how data analytics and visualization can help businesses track performance, discover trends, and improve strategic planning in the luxury watch market.
